# TurtleBot3 GUI

Графический интерфейс для управления TurtleBot3 с использованием PyQt5.

## Функциональность

- Управление движением робота с помощью кнопок (вперед, назад, влево, вправо, стоп)
- Отображение данных с датчиков:
  - Состояние батареи (напряжение и процент заряда)
  - Позиция и ориентация (одометрия)
  - Данные с лазерного сканера (минимальное и максимальное расстояние)
  - Показания энкодеров колес

## Установка зависимостей

```bash
sudo apt install python3-pyqt5
pip3 install rospkg
```

## Запуск

### Вариант 1: Использование launch файла

```bash
roslaunch turtlebot3_gui turtlebot_gui.launch
```

### Вариант 2: Прямой запуск скрипта

```bash
cd /home/vanya890/catkin_ws/src/turtlebot3_gui/src
python3 turtlebot_gui.py
```

## Использование

1. Запустите симуляцию TurtleBot3 в отдельном терминале:
   ```bash
   export TURTLEBOT3_MODEL=burger
   roslaunch turtlebot3_gazebo turtlebot3_empty_world.launch
   ```

2. Запустите GUI приложение (один из способов выше)

3. Используйте кнопки управления для перемещения робота:
   - Forward: движение вперед
   - Backward: движение назад
   - Left: поворот на месте влево
   - Right: поворот на месте вправо
   - STOP: немедленная остановка

4. Наблюдайте за данными с датчиков в верхней части окна
