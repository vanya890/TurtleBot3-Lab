## Управление TurtleBot3 через консоль

Вы спрашиваете о том, как управлять TurtleBot3 с помощью клавиатуры через консоль.<cite />

## Запуск телеоперации с клавиатуры

Для управления роботом через консоль используется пакет `turtlebot3_teleop`, который позволяет управлять движением робота с помощью клавиш клавиатуры.<cite />

### Для симуляции в Gazebo

**Терминал 1: Запустите симуляцию**
```bash
export TURTLEBOT3_MODEL=burger
roslaunch turtlebot3_gazebo turtlebot3_empty_world.launch
``` [1](#6-0) 

**Терминал 2: Запустите управление с клавиатуры**
```bash
roslaunch turtlebot3_teleop turtlebot3_teleop_key.launch
```
<cite />

Или используйте команду напрямую:
```bash
rosrun turtlebot3_teleop turtlebot3_teleop_key
```
<cite />

### Для AutoRace в симуляции

Если вы хотите управлять роботом в мире AutoRace:<cite />

**Терминал 1: Запустите мир AutoRace**
```bash
roslaunch turtlebot3_gazebo turtlebot3_autorace.launch
``` [2](#6-1) 

**Терминал 2: Запустите телеоперацию**
```bash
ros2 run turtlebot3_teleop teleop_keyboard
``` [3](#6-2) 

Для ROS Noetic используйте:
```bash
rosrun turtlebot3_teleop turtlebot3_teleop_key
```
<cite />

### Управление клавишами

После запуска телеоперации вы увидите инструкции по управлению в терминале:<cite />
- `W` - движение вперед
- `X` - движение назад  
- `A` - поворот влево
- `D` - поворот вправо
- `S` - остановка
- `Space` - экстренная остановка

## Notes

Пакет `turtlebot3_teleop` должен быть установлен вместе с основными пакетами TurtleBot3.<cite /> Если вы следовали инструкциям по установке из предыдущих сообщений, этот пакет уже должен быть установлен.<cite /> Телеоперация работает как в симуляции Gazebo, так и с реальным роботом.<cite />
